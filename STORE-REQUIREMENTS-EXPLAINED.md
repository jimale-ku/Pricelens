## üì∏ Screenshot Requirements

### Google Play Store:

**Sizes needed:**
- Phone: 16:9 or 9:16 aspect ratio
- Tablet: 16:9 or 9:16 aspect ratio
- Minimum: 320px width
- Maximum: 3840px width

**How to take screenshots:**
1. Run app on real device
2. Navigate to key screens
3. Take screenshots (device screenshot button)
4. Edit if needed (add text, highlights)
5. Upload to Play Console

**What to show:**
- Home screen
- Main features
- Category pages
- Search results
- Profile/settings

### Apple App Store:

**Sizes needed:**
- iPhone 6.7": 1290 x 2796 pixels
- iPhone 6.5": 1284 x 2778 pixels
- iPhone 5.5": 1242 x 2208 pixels
- iPad 12.9": 2048 x 2732 pixels
- iPad 11": 1668 x 2388 pixels

**How to take screenshots:**
1. Use Xcode Simulator or real device
2. Take screenshots of key screens
3. Use same screenshots for all sizes (resize)
4. Upload to App Store Connect

**What to show:**
- Home screen
- Main features
- Key functionality
- Beautiful UI

## üì± How to Take Screenshots

### Android:

1. **On device:**
   - Open app
   - Navigate to key screens
   - Press Power + Volume Down
   - Screenshot saved to gallery

2. **Or use Android Studio:**
   - Open emulator
   - Use screenshot tool
   - Save images

### iOS:

1. **On device:**
   - Open app
   - Navigate to key screens
   - Press Power + Home (or Power + Volume Up)
   - Screenshot saved to Photos

2. **Or use Xcode Simulator:**
   - Open simulator
   - Use Device ‚Üí Screenshot
   - Save images
